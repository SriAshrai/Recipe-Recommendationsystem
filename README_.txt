## Dataset
This project uses the **Food.com Recipes and Reviews** dataset, available on Kaggle. The dataset contains information about recipes, their ingredients, cooking times, and user reviews.

### How to Download
1. Visit the [Food.com Recipes and Reviews dataset page on Kaggle](https://www.kaggle.com/datasets/).
2. Download the dataset files (`RAW_recipes.csv` and `RAW_interactions.csv`).
